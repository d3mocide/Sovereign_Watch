# 2026-03-28 - Release v0.57.0 OSINT Reader and Widget Refinement

## Issue
A substantial set of Intel/Tactical OSINT workflow improvements had been implemented, but release metadata still reflected the previous version and did not summarize the new reader-mode and widget UX changes.

## Solution

- Updated `frontend/package.json` version from `0.56.0` to `0.57.0`.
- Updated `CHANGELOG.md` with a new `0.57.0` section covering:

  - Tactical/Intel reader parity

- Replaced `RELEASE_NOTES.md` with release notes for `v0.57.0`.
- Preserved previously created per-task implementation notes under `agent_docs/tasks/`.

  - `cd frontend && pnpm run test`

## Benefits
- Produces a coherent release artifact set for the completed OSINT workflow work.
- Makes the new in-app reader and widget changes visible in release documentation.
- Prepares the repository for tagging and push as `v0.57.0`.
