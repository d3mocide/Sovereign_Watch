# Release - v0.36.1 - AI Analyst Formatting Fix

## High-Level Summary

This patch release resolves a critical visual formatting issue in the AI Analyst panel's clipboard functionality. It ensures that tactical assessments generated by the AI engines project the same "Premium" professional structure when copied and pasted into external documents as they do in the Sovereign Watch interface.

## Key Features

- **Standardized Intelligence Formatting**: Implemented a robust text normalization engine specifically for intelligence reports.
- **Copy-Paste Fidelity**: Resolved the "one word per line" bug when pasting into Notepad and other plaintext editors.
- **SSE Stream Sanitization**: Enhanced the frontend's handling of Server-Sent Events (SSE) to strip low-level carriage returns that were causing invisible formatting breaks.

## Technical Details

- **Utility Refactoring**: Extracted character cleaning logic into a shared `formatAnalysisText` helper.
- **Regex Improvements**: Enhanced cleaning patterns to handle smart quotes, punctuation spacing, and header normalization.
- **Stream Patch**: Stripped `\r` characters at the `useAnalysis` hook level to prevent state pollution.
